sorting algorithms
我一共比较了八种排序方法，实验的数据规模是10000，100000，500000,1000000,5000000。
前四种数据规模的排序，每种排序算法都各执行了三遍，最后5000000（五百万）数据量每种排序方法只执行了一遍。（由于本人电脑没有SSD，也没有GPU，跑程序断断续续跑了四五天……）
产生随机数的程序为 genda.txt
产生的随机数数据保存在了 data.txt 中
各种排序方法排除来的结果保存在了 out_*Sort.txt 中
跑出来的第一手结果（执行时间）保存在了 demo_Realtime.txt 中
用于画图的处理后的数据保存在了 data_out.txt 中
处理所有程序及计算执行时间的脚本为 process.sh （项目的执行过程在其中都有展示）
画图的程序为 plot.py (NumPy, matplotlib, python)
画出来的图像为 plot.jpg

工程中plot.py内用于绘图的数据是手工录入的……
其他的步骤均可通过 bash process.sh 一条指令完成。
